Remarks

Code must NOT be localized. It is required to universally identify a particular message. Text should be localized if the source tool supports localization.

Diagnostic codes (Code) are of the form <prefix><number> (example: FOO01234). This is a unique identifier for the message that allows a user to reference it for support or other diagnostic analysis. The <prefix> portion of the code indicates the application source of the message.
