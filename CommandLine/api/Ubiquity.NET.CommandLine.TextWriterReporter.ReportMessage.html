ReportMessage(MsgLevel, string)

Virtual method to report a message formatted as a string

protected virtual void ReportMessage(MsgLevel level, string msg)

Parameters

level MsgLevel

Level of the message

msg string

Message formatted as a string

Remarks

The default base implementation will simply redirect messages based on level to the TextWriter writer for level. It is possible that such a writer is Null and the messages go nowhere.

Exceptions

InvalidOperationException

Invalid/Unknown level - should never hit this, internal or derived type error if it does.
