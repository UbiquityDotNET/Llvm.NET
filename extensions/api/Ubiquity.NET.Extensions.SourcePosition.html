Remarks

Positions use a Line numbering similar to most editors that is 1 based.([1..n][0 = uninitialized/unknown]). Column position, also like most editors, use a 0 based value ([0..n-1]). The index, if available, indicates the 0 based index into a theoretical vector of characters for the position. Not all location information includes such a thing so it may not have a value.
