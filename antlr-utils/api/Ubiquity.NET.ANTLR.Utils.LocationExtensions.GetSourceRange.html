Attempts to retrieve the SourceRange from a Antlr4.Runtime.RuleContext

public static SourceRange GetSourceRange(this RuleContext ctx)

Parameters

ctx RuleContext

Context to get the span from

Returns

SourceRange

SourceRange for this input context or a default constructed one

Remarks

Not all Antlr4.Runtime.RuleContext derived types will support line+col location information. This only tests for a Antlr4.Runtime.ParserRuleContext and retrieves the location from that. The base RuleContext and other derived types simply get a default constructed location as the location is not known. (They only store location as an integral interval without the line+col information)
