Remarks

important

For runtimes prior to .NET 5 the behavior of TrimEntries is emulated here. The implementation of that emulation is not as performant as the official form in later runtimes. This emulation was chosen for correctness of behavior and simplicity of implementation over performance. If absolute best performance is desired then use the latest runtime.
