Remarks

This is generally used by language specific extensions that will also emit the comment leading/trailiing text as needed. Thus it is a general language neutral facility that is used to produce the final language specific comments. It will perform the following on the input string:

  1. EscapeComment(string) to ensure character escaping is applied to the whole string
  2. SplitLines(string, StringSplitOptions2)to split the string into distinct lines
  3. EscapeForXML(IEnumerable<string>) to ensure the lines are valid for XML doc comments
  4. SkipDuplicates(IEnumerable<string>) to fold duplicate entries into one (usually to reduce multiple blank lines to one)
