Interface for implementing the Visitor pattern with IAstNode

public interface IAstVisitor<out TResult>

Type Parameters

TResult

Result type of the visit

Remarks

This interface is used for visiting an AST, typically for generating code but may also be used to detect errors in the AST etc..

The TArg is typically used for a Byref-like type where the type may NOT be stored on the heap and MUST be passed via `ref readonly`.

Methods
