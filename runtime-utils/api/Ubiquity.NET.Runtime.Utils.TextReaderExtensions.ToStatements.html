ToStatements(TextReader, Action<ReadyState>?, CancellationToken)

Async operator to encapsulate conversion of text from a TextReader into an observable sequence of Kaleidoscope statements

public static IAsyncEnumerable<string> ToStatements(this TextReader reader, Action<ReadyState>? prompt, CancellationToken cancelToken = default)

Parameters

reader TextReader

Input reader

prompt Action<ReadyState>

Action to provide prompts when the transform requires new data from the reader

cancelToken CancellationToken

Cancellation token for the async operation

Returns

IAsyncEnumerable<string>

Async sequence of complete statements ready for parsing

Remarks

This uses a common statement completion character of ';' if that is not desired then the ToStatements(TextReader, Action<ReadyState>?, char, CancellationToken) overload takes a parameter for the completion character.
