Remarks

While default interface methods seems like a great idea, it's not yet complete enough to be useful. In particular there's pretty much no debugger support for evaluating such things, leaving you with no way to see what they produce when used as a property. Hopefully, that will be resolved in the future - but for now it is more a hindrance than it is a help.
