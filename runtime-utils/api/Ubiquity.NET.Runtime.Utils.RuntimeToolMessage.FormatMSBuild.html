Parameters

origin string

Origin of the message (May be a file or app name)

location SourceLocation?

Location in origin of the message source (If origin is a file (do not use if origin is not a path)

subcategory string

Sub category of the message normally omitted but may include additional information about the message

category MsgCategory

Category of the message; Most tooling only deals with Warning or Error

code string

Code for the message - must not contain spaces

msgText string

Text of the message [Localized]

Format provider for formatting the values
