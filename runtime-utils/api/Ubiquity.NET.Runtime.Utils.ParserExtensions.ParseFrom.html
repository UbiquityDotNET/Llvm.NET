Parse results as an IAstNode

Type Parameters

T

Type implementing IParser

Remarks

If the parse fails then the result is null. Errors from the parse are reported through error listeners provided to the parser. Normally, this is done via the constructor of a type implementing IParser.

important

If the parse succeeds, but creation of the AST fails, then the result is an AST tree with some nodes as ErrorNode
