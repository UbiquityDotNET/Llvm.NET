CreateAttribute(LazyEncodedString, uint, ulong[], ulong[])

Create an attribute that accepts a constant range value

AttributeValue CreateAttribute(LazyEncodedString name, uint numBits, ulong[] lowWords, ulong[] upperWords)

Parameters

name LazyEncodedString

name of the attribute

numBits uint

Number of valid bits in the range values

lowWords ulong[]

Array of bits composing the low end value of the range

upperWords ulong[]

Array of bits composing the upper end value of the range

Returns

AttributeValue

Attribute value

Remarks

A constant range, as the name implies, contains a range of constant values that are a lower bound and upper bound for a range. The values of each end are an arbitrary precision integer (APInt) and thus numBits indicates the total bit width of the values in the range.

Exceptions

ArgumentException

The name does not accept a constant range value
