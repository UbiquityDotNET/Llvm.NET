true if the attribute has a parameter
Remarks
Most of the attributes are simple boolean flags, however some, in particular those dealing with sizes or alignment require a parameter value. This method is used to determine which category the attribute falls into. Any attribute returning true requires an integer parameter and has special handling so cannot be used in the AddAttributes() methods.
