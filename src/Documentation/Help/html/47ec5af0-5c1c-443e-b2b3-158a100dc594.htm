Welcome to Llvm.NET

[This is preliminary documentation and is subject to change.]

Llvm.NET is a managed wrapper around an extended LLVM-C API including an OO model that closesly matches the underlying LLVM internal object model. This allows for building code generation and other utilities leveraging LLVM from .NET applications.
