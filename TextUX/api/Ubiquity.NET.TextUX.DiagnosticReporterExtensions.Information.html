Information(IDiagnosticReporter, string?, SourceRange, Uri?, string?, IFormatProvider?, string, params object[])

Reports a Information level message to self

public static void Information(this IDiagnosticReporter self, string? code, SourceRange location, Uri? origin, string? subCategory, IFormatProvider? formatProvider, string fmt, params object[] args)

Parameters

self IDiagnosticReporter

Reporter to report message to

code string

Identifier code for this message

location SourceRange

Location in the origin that this message refers to

origin Uri

Origin of the diagnostic (Usually the origin is a File, but may be anything or nothing)

subCategory string

Subcategory for this message

formatProvider IFormatProvider

Format provider to use when formatting the string

fmt string

Format string for the message

args object[]

Arguments for the message

Remarks

The reporter (self) may filter out any messages reported for this level. The fmt is NOT applied to args unless the level for the message is enabled. This helps reduce the overhead of producing the final formatted string if it is ignored anyway.
