Parameters

level MsgLevel

Level of messages to enable for this reporter [Default: Information

colorMapping ImmutableDictionary<MsgLevel, AnsiCode>

Provides the color mapping for the message levels this reporter will use (see remarks)

Remarks

colorMapping (or a default if null) is set as the ColorMap property.

The default colors, if not provided via colorMapping, are:

LevelDescription
Verbose AnsiCodes.Color.LtBlue
Information AnsiCodes.Color.Default
Warning AnsiCodes.Color.LtYellow
Error AnsiCodes.Color.LtRed

Any level not in ColorMap is reported using AnsiCodes.Color.Default.
